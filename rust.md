# Rust шпаргалка

https://rurust.github.io/rust_book_ru/getting-started.html


## Простой способ создать новый Cargo проект

`cargo new hello_world --bin`

## Сборка

`cargo build --release`

`Cargo.toml`

```toml
[package]

name = "hello_world"
version = "0.0.1"
authors = [ "Your name <you@example.com>" ]
```
